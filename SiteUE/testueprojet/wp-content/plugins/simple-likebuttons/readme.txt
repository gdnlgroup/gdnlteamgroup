Adds a facebook, Google+ and Twitter button to your posts and pages with one click.

== Description ==

The plugin does what it should: adds to posts and pages the classic facebook, google+ and twitter buttons. Very easy to install and no initial configuration needed.

== Installation ==

1. Upload the plugin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate / deactivate icons if needed in the settigs panel
